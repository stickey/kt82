# Favicon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the default Vite placeholder favicon with a green KT82-branded SVG favicon across all four apps, and fix the two apps that still have the generic Vite page title.

**Architecture:** Each Vite app has its own `public/` directory served as static assets, and an `index.html` at its root. We add `favicon.svg` to each `public/`, update the `<link rel="icon">` in each `index.html`, delete the existing `vite.svg`, and fix the `<title>` tags for Captain and Manager.

**Tech Stack:** SVG, HTML, Vite (no build step required — SVG is served as-is from `public/`)
